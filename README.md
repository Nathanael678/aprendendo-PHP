# PHP Estudos

Repositório dedicado ao desenvolvimento das minhas habilidades em PHP, com foco na construção de uma base sólida para atuação na área de desenvolvimento web.

## Sobre

Sou estudante da área de Tecnologia da Informação e este repositório documenta minha evolução prática no aprendizado de PHP. Aqui concentro exercícios, testes e pequenos exemplos aplicados, com o objetivo de consolidar conceitos fundamentais e boas práticas.

## Objetivo profissional

Busco oportunidade de estágio ou posição inicial na área de desenvolvimento, onde eu possa aplicar meus conhecimentos, aprender com profissionais experientes e evoluir tecnicamente.

## Conteúdo abordado

* Estrutura básica de aplicações PHP
* Integração entre PHP e HTML
* Saída de dados (`echo`, `print`)
* Funções nativas (ex: `phpinfo()`)
* Execução em ambiente local com Apache (XAMPP)

## Tecnologias utilizadas

* PHP
* HTML
* Apache (XAMPP)

## Como executar o projeto

1. Instalar o XAMPP
2. Iniciar o módulo Apache
3. Inserir os arquivos na pasta `htdocs`
4. Acessar no navegador:

```
http://localhost:8080/
```

ou:

```
http://localhost/
```
